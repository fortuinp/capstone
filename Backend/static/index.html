Router Information

General Routing
Routes HTTP Methods Description
/ GET To display the home page.
User Routing
Routes HTTP Methods Description
/users/ GET To display all users.
/users/:id/ GET To display a user.
/users/register/ POST To add a new user.
/users/update/:id/ Patch To update a user info.
/users/delete/:id/ Delete To delete a user.
Product Routing
Routes HTTP Methods Description
/products/ GET To display all Products.
/products/:id/ GET To display a Product.
/products/addProduct/ POST To add a new Product.
/products/update/:id/ Patch To update a product info.
/products/delete/:id/ Delete To delete a product.
